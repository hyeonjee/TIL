### Django 디렉토리(폴더) 생성
: 터미널에서 `mkdir 이름`

<br>

### vscode에서 터미널 창 띄우기
: `control + shift + '`

<br>

### 가상환경 설치
`pip3 install virtualenv`  
: python 버전 관리와 패키지 충돌 방지를 위해

### 가상환경 폴더 생성
`virtualenv venv`

### 가상환경 실행
`source venv/bin/activate`

<br>

### Django 설치
`pip install django`

### Django 프로젝트 생성
`django-admin startproject 이름`

<br>

### 서버 실행 명령어
`python manage.py runserver`

### 서버 종료하기
`control + c`
